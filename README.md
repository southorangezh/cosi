# COSI
---

Cloud object storage integrate


* Integrate Alibaba Cloud\Huawei Cloud\Tencent Cloud Object Storage to upload static files

# Install

```

npm install cosi --save

```
# Command

```
npm run cosi:test
```

# Use environment

Node.js
